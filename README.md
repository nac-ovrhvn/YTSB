YouTube and other social media platforms have curated algorithms that trap you in an infinite loop of content consumption.

This browser extension helps to solve that by keeping the user mindful of the time they are spending on the platform.

![YTSB popup](screenshot.PNG)

Download Instructions:

Firefox:
1) Go to about:debugging#/setup
2) Click on 'This Firefox' tab
3) Load Temporary Add-on
4) Select the zip file

Chrome:
1) Go to extensions page and enable developer mode
2) Click on 'Load unpacked'
3) Select the unpacked zip folder
